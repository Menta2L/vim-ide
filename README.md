vim-ide
=======

Features
--------

* Code completion
* Command T for project file search
* Many colorschemes
* NERDTree file browsing
* Clear trailing whitespace

Included colorschemes:

- Tomorrow
- Tomorrow-Night
- Tomorrow-Night-Blue
- Tomorrow-Night-Bright
- Tomorrow-Night-Eighties
- molokai
- solarized (dark and light)
- vividchalk

Supported languages:

* Markdown
* Node.js
  - JSHint
* Python
  - Pylint

Requirements
------------

- git
- ruby
- gcc
